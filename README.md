# COAL Project: DUCK SHOOT

A repository containing code for the "DUCK SHOOT" game.

## Game Overview

The game features:
- One title page
- One main page
- Two game options:
  1. One Duck
  2. Two Ducks

## Game Options

### One Duck

In the One Duck option, the player has 2 tries to shoot a duck. Each time a duck is killed, the score is updated and printed.

### Two Ducks

In the Two Ducks option, there are two levels:

#### Level 1: Two Ducks

- Contains two ducks that move randomly in different directions.
- The player's task is to shoot the ducks.
- The score is updated and printed with each successful shot.

#### Level 2: Speed Duck

- Introduces a special brown duck with a faster speed than the regular yellow duck.
- The brown duck awards more points than the yellow duck.

## Getting Started

To get started with the game, clone the repository and run the main game script. Ensure you have the necessary dependencies installed.

## How to Play

1. Launch the game.
2. Select the option "Play" by pressing 1 or choose "Exit" by pressing 0.
3. Select either "One Duck" or "Two Ducks" option from the main menu by pressing 1 or 2 respectively.
4. Follow the on-screen instructions to aim and shoot the ducks.
5. Try to achieve the highest score!
